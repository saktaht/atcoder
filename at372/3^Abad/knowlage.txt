3の何乗で割り切れるか考える

一番大きい３のi乗で一回割る

そのあまりをまた3のi上で割るを繰り返す

chatGPTの例
例えば、M = 10 を入力したとします。

1回目のループ (k = 0)

M % 3 は 10 % 3 = 1 です。
A += [0] * 1 なので、A に 0 を1回追加します。A = [0]
次に M //= 3 で M = 10 // 3 = 3 になります。
2回目のループ (k = 1)

M % 3 は 3 % 3 = 0 です。
A += [1] * 0 なので、何も追加しません。A = [0]
次に M //= 3 で M = 3 // 3 = 1 になります。
3回目のループ (k = 2)

M % 3 は 1 % 3 = 1 です。
A += [2] * 1 なので、A に 2 を1回追加します。A = [0, 2]
次に M //= 3 で M = 1 // 3 = 0 になります。
4回目以降のループ (k = 3 から 10 まで)

M は既に0なので、M % 3 は常に0になります。したがって、何も A に追加されません。
